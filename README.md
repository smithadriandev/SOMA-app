# Soma

Frontend Flutter do **Sistema de Organização e Monitoramento do Alzheimer**.

O app usa apenas Flutter com Material, dados mockados em memória e asset local para a logo.

## Fluxo

- Splash com logo.
- Login com validação de email terminado em `.com`.
- Cadastro com validação de senha e CPF.
- Recuperação de senha com atualização no mock em memória.
- Home simples após login.
